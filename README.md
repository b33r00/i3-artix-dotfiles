# Artix Linux i3 Rice

My personal **Artix Linux (runit)** rice focused on minimalism, performance, and a dark, clean aesthetic.
Built around **i3wm** with lightweight tools and custom configuration.

## Environment
- **OS:** Artix Linux (runit)
- **Window Manager:** i3wm
- **Bar:** polybar
- **Compositor:** picom
- **Launcher:** rofi
- **Terminal:** alacritty
- **Shell:** zsh
- **Lockscreen:** mantablockscreen

## Theme & Appearance
- **GTK Theme:** Adwaita
- **Cursor Theme:** WinSur-dark-cursors
- **Style:** Dark / minimal with subtle accent colors
